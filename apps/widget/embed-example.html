Quick Start

Add this single line to your website to embed the chatbot widget:

<script src="https://yourcdn.com/widget.js" data-restaurant-slug="demo-bistro" data-position="bottom-right" data-primary-color="#667eea" data-brand-name="Demo Bistro"></script>

Configuration Options

Attribute Type Required Description
data-restaurant-slug string Required Your restaurant's unique identifier
data-position string Optional Position: bottom-right, bottom-left, top-right, top-left
Default: bottom-right
data-primary-color string Optional Theme color (hex format)
Default: #667eea
data-brand-name string Optional Display name in chat header
Default: Uses slug
data-api-url string Optional API endpoint URL
Default: http://localhost:3000
data-z-index number Optional CSS z-index value
Default: 9999

Advanced: Programmatic Control

For advanced use cases, you can control the widget via JavaScript:

// Initialize manually window.RestaurantWidget.init({ restaurantSlug: 'my-restaurant', position: 'bottom-right', primaryColor: '#667eea', brandName: 'My Restaurant', apiUrl: 'https://api.example.com', zIndex: 9999 }); // Destroy instance window.RestaurantWidget.destroy('restaurant-widget-my-restaurant'); // Check active instances console.log(window.RestaurantWidget.instances);
